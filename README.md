# e-iot

简单，高性能，基于 vert.x 实现多种充电协议。

Simple, high performance, based on vertx to implement a variety of charging protocols

### 实现的协议

* 云快充协议1.x  (eiot-ykc) doing
* ocpp1.6j  (eiot-ocpp1) doing
* ocpp 2.0.1 (eiot-ocpp2) doing
